Vim 高级技能书

刚上手Vim没多久，除了基本的移动方向、插入、删除、修改等在tutor中学到的技能以外，我决定把遇到的高级技能不断更新在这个文档里，忘了可以拿出来看看，也可以当作一个教程。

功能类

包括Vim本身知道的一些功能，比如多标签。

标签页

  • 新建标签页

  • 对所有标签进行操作

  • 在标签页之间切换

批量操作类

包括批量删除列、批量条件替换等。

批量操作列

  • 批量在所有行首添加内容

  • 批量删除行首内容

  • 批量在行尾添加内容

  • 批量在行尾删除内容

批量替换

:%s/old/new/g

%表示匹配全部行，即纵向所有行进行操作

s/old/new表示进行的是替换操作，用new替换old，old可以使用正则表达式

/g表示匹配全部列，即横向对所有列进行操作

以上合起来即对全文进行替换操作把所有符合old的东西替换成new
